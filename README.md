# TFF-portal

Statisk webbportal (HTML/CSS) med tre sidor:
- `index.html` (Start/portal)
- `guides.html` (Guider/sök)
- `utbildningar.html` (Utbildningar)

## Publicering (GitHub Pages)
Settings → Pages → Deploy from a branch → `main` → `/ (root)`.

## Informationssäkerhet
Innehåller inga personuppgifter och ingen inloggning. Avsedd som demo/prototyp.
